##### HTML
-태그를 이용해 구조적으로 작성된 마크업 언어

-W3C: 웹의 표준화를 추진하는 곳

```html
<!-- 글자의 포맷 -->
<meta charset="UTF-8">
<!-- viewport에 대한 정의. 스크린의 너비를 모두 사용 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```
