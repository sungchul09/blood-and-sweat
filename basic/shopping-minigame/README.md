### Mobile First

디자인, 웹사이트를 만들때 모바일을 먼저 만들고 어떻게 확장해서 반응형으로 만들지 고민하는 것.

### 레이아웃 구조 분석하기

컨텐츠가 유동적으로 쭉 흐르도록 하다가 정말 공간 활용이 비효율적일때 끊어서 레이아웃을 재배치하는것이 좋다.

### WireFrame

디자이너가 상세하게 디자인 하기전 전반적으로 어떻게 구조를 잡아나가야할지 구조를 크게 나누어서 박스단위로 와이어프레이밍 한다.라고한다.

### MDN 사용법

html reference로 이동 -> 원하는 tag 선택 -> Attributes 에서 속성 설명 확인 -> Browser compatibility에서 지원 가능한 브라우저 확인
-> Consistent하게 디자인하기 -> 모든 디자인은 일정하게 디자인하는것이 좋다.

### CSS 초기 셋팅 (\* 세팅)

padding:0; margin: 0; box-sizing: border-box; (테두리를 포함하여 size를 측정한다)
